# ComicMTC_v2
<h1 align="center">ComicMTC_v2_V2</h1>

## 简介
　在线漫画阅读器,项目采用 Kotlin+MVVM+LiveData+协程+Retrofit. 接口来自有妖气,以过滤vip的章节，此开源项目仅做学习交流使用。

如果对你有帮助不妨start支持一下.感谢ヽ(￣ω￣(￣ω￣〃)ゝ
